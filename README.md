# fable5_test_01

Interactive WebGL landing page — "Ankward Corp." moss-growing hero.

## 🌿 바로 보기 (실행 링크)

**https://lgw7126.github.io/fable5_test_01/**

> 위 링크가 열리지 않으면 GitHub Pages가 아직 꺼져 있는 거예요.
> [여기를 클릭](https://github.com/lgw7126/fable5_test_01/settings/pages)해서
> **Branch**를 `None` → `main`으로 바꾸고 **Save**를 누른 뒤 1~2분 기다리면 열립니다.

## 사용법

- 나뭇가지 위에서 **마우스를 움직이면** 이끼가 자라납니다
- **스크롤(휠)** 로 화면을 확대/축소할 수 있습니다

## 내 컴퓨터에서 직접 실행하기

```sh
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/index.html 열기
```

Built with Three.js (vendored in `vendor/`), shell-textured fur shaders for
the moss, and a paintable canvas mask driven by raycasting.
